Калибровочный график и концентрация белка

Проведите прямую через точки. Затем по ОП = 0.193 (лунка А7) и ОП = 0.326 (А8) найдите соответствующие значения мкг белка.
Расчёт концентрации белка

Из уравнения прямой (y = kx + b) найдите x при ОП = 0.193 (А7, 25 мкл липазы) и ОП = 0.326 (А8, 50 мкл).

Конц. белка (из А7, 25 мкл): мкг / 25 мкл
Конц. белка (из А8, 50 мкл): мкг / 50 мкл
Концентрация в препарате: мг/мл
Уравнение прямой: ОП = 0.00075 × мкг + 0.060
Из А7 (ОП = 0.193): мкг = (0.193 − 0.060) / 0.00075 = 177 мкг в 25 мкл → 7.1 мг/мл
Из А8 (ОП = 0.326): мкг = (0.326 − 0.060) / 0.00075 = 355 мкг в 50 мкл → 7.1 мг/мл
Концентрация белка ≈ 7.1 мг/мл

Задание 4

Активность липазы в препарате фермента

5 баллов ▼
Рассчитайте активность липазы в мкмолях нейтрального жира / мин / мг белка, используя данные из заданий 2 и 3.

Исходные данные: к 500 мл эмульсии добавлено 250 мкл препарата липазы. Аликвоты — 1 мл каждая.
Расчёт активности
Алгоритм (используем только Пробу 1 — 5 мин, т.к. к 10 мин наступает плато):
1. c(СЖК) = c(Пробы 1) из задания 3, например 21.0 мМ
2. n(СЖК) в 500 мл = c × 0.5 л × 1000 = c × 500 [мкмоль]
3. n(TAG) = n(СЖК) / 3 — каждый триацилглицерид даёт 3 жирных кислоты
4. Скорость = n(TAG) / 5 мин [мкмоль ТАГ / мин]
5. m(белка) = c(белка) × 0.25 мл = 7 × 0.25 = 1.75 мг
6. Уд. активность = Скорость / m(белка) ≈ 400 мкмоль/мин/мг
n(СЖК) в 500 мл = c(П1) × 500: мкмоль
n(TAG) = n(СЖК) / 3: мкмоль
Скорость = n(TAG) / 5 мин: мкмоль ТАГ/мин
Масса белка (7 мг/мл × 0.25 мл): мг
Удельная активность: мкмоль ТАГ / мин / мг белка
При c(П1) = 21.0 мМ (из задания 3):
n(СЖК) = 21.0 × 500 = 10 500 мкмоль
n(TAG) = 10 500 / 3 = 3 500 мкмоль
Скорость = 3 500 / 5 = 700 мкмоль ТАГ/мин
m(белка) = 7 мг/мл × 0.25 мл = 1.75 мг
Уд. активность = 700 / 1.75 = 400 мкмоль ТАГ/мин/мг
Задание 5

Молекулярная масса нейтрального жира

5 баллов ▼
Навеску жира массой 1.21 г подвергли полному щелочному гидролизу в 100 мл 100 мМ NaOH. После гидролиза концентрация непрореагировавшего NaOH = 55 мМ. Рассчитайте молекулярную массу нейтрального жира.
Расчёт молекулярной массы
Ход рассуждений:
Нейтральный жир (триацилглицерид) реагирует с 3 молями NaOH:
TAG + 3 NaOH → глицерол + 3 карбоксилат Na⁺

n(NaOH исходное) = c × V = 100 мМ × 0.1 л = 10 ммоль
n(NaOH после) = 55 мМ × 0.1 л = 5.5 ммоль
n(NaOH прореагировало) = 10 − 5.5 = 4.5 ммоль
n(TAG) = n(NaOH) / 3 = 1.5 ммоль = 0.0015 моль
M = m / n = 1.21 г / 0.0015 моль = ?
n(NaOH исходное) = 0.1 л × 100 мМ: ммоль
n(NaOH после) = 0.1 л × 55 мМ: ммоль
n(NaOH прореагировало): ммоль
n(TAG): ммоль
Молекулярная масса TAG: г/моль
n(NaOH₀) = 0.1 л × 0.1 моль/л = 0.01 моль = 10 ммоль
n(NaOH после) = 0.1 л × 0.055 моль/л = 0.0055 моль = 5.5 ммоль
n(NaOH потрачено) = 10 − 5.5 = 4.5 ммоль
n(TAG) = 4.5 / 3 = 1.5 ммоль = 0.0015 моль
M = 1.21 г / 0.0015 моль = 807 г/моль

Для сравнения: триолеин (C₅₇H₁₀₄O₆) M = 885 г/моль; трипальмитин (C₅₁H₉₈O₆) M = 807 г/моль ✓
